# Tmux Keybinds

## Prefix
`Ctrl + Space`

## New Tmux Session
`Prefix + c`

## Move Between Sessions
* Next Session
    `Prefix + n`
* Previous Session
    `Prefix + p`
* Navigate to specific session
    `Prefix + {1,2,3,...,9}`

## Split Windows
* Horizontal Split
    `Prefix + %`

* Vertical Split
    `Prefix + -`
